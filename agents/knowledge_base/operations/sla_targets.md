# SLA Targets

## Issue Response Times

| Issue Type | First Response | Resolution Target |
|:---|:---|:---|
| Security | < 4 hours | < 24 hours |
| Bug (critical) | < 12 hours | < 48 hours |
| Bug (normal) | < 24 hours | < 1 week |
| Feature request | < 48 hours | Roadmap review |
| Question | < 24 hours | Same response |
| Documentation | < 48 hours | < 1 week |

## PR Review Times

| PR Type | QA Validation | Human Review |
|:---|:---|:---|
| Engineer (minor) | < 1 hour | < 1 week |
| Engineer (major) | < 1 hour | Next weekend review |
| External contributor | < 24 hours | < 1 week |
