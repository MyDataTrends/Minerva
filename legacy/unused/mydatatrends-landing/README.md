# MyDataTrends Landing Page

This repository contains a static landing page for **MyDataTrends** built with Tailwind CSS. It showcases product benefits, pricing tiers, and testimonials. The site is fully responsive and can be deployed to GitHub Pages or Netlify.

## Development

1. Install dependencies (optional if using Tailwind CDN):
   ```bash
   npm install
   ```
2. Build Tailwind (if not using the CDN):
   ```bash
   npm run build
   ```
3. Open `index.html` in your browser to view the site.

## Deployment

The page is static, so it can be served via any static host. To deploy on GitHub Pages, push this folder to a repository and enable Pages in the repo settings.
